Complete the following steps to create a new storage bucket.
Using Cloud Console:
  1. In the Google Cloud Console, go to the Cloud Storage Browser page.
  2. Click Create bucket.
  3. On the Create a bucket page, enter your bucket information.
    • For Name your bucket, enter a globally unique name. For example:
    • For Choose where to store your data, select a Location type and Location where the bms servers created.
  4. Click Create
  5. In the list of buckets, click on the name of the bucket that you created in previous steps.
  6. In the Objects tab for the bucket, either:
    • Drag and drop the desired files from your desktop or file manager to the main pane in the Cloud Console.
    • Click the Upload Files button, select the files you want to upload in the dialog that appears, and click Open.
Using gsutil: Notes:
  • You cannot upload more than 5TiB per upload request
  • Consider using of resumable upload, it can save your time if network failure occurs
